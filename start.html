تشغيل اللعبة

— كودك

طريقة التشغيل

  1. ثبّت تطبيق Microsoft PowerPoint على جهازك.
  2. افتح التطبيق وسجّل دخولك بحساب Microsoft (مجاني).
  3. اضغط زر تحميل اللعبة ثم افتح الملف من تطبيق الملفات (Files) باستخدام PowerPoint.
  4. داخل PowerPoint اختر وضع العرض/Slide Show لبدء اللعب.
تحميل اللعبة (PPTX)
إذا زر “فتح اللعبة” ما اشتغل عندك، استخدم “تحميل اللعبة” ثم افتحها يدويًا من الملفات.
تحميل PowerPoint للآيفون والآيباد (App Store) تحميل PowerPoint للأندرويد (Google Play) تغيير الكود / إعادة التفعيل
ملاحظة: بعض الأجهزة تمنع فتح الملف مباشرة عبر الزر، لذلك وضعنا زر تحميل كخيار مضمون.
